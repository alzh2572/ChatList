Возможности

  • Параллельная отправка промта во все активные модели
  • Таблица результатов с выбором строк для сохранения
  • Управление моделями, промтами и историей
  • AI-ассистент для улучшения промтов
  • Экспорт в Markdown и JSON
  • Светлая и тёмная тема, настройка размера шрифта

Быстрый старт

  1. Скачайте и установите ChatList-*-setup.exe из последнего Release.
  2. Получите API-ключ на OpenRouter.
  3. Создайте файл .env рядом с программой или в каталоге запуска:
    OPENROUTER_API_KEY=sk-or-v1-ваш-ключ
  4. Запустите ChatList, введите промт и нажмите «Отправить».

Для запуска из исходников: Python 3.11+, pip install -r requirements.txt, python main.py.
